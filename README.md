## Overview

Mind Game is an engaging memory card matching game built with vanilla JavaScript. Test your memory by matching pairs of popular meme cards featuring characters like Shrek, Mr. Bean, Cristiano Ronaldo, and more. The game challenges players to remember card positions and make matches with as few moves as possible.

## Features

- **Classic Memory Game Mechanics**: Flip cards to find matching pairs
- **Meme Collection**: Features popular internet memes and personalities
- **Move Counter**: Track your performance with a move counter
- **Responsive Design**: Play seamlessly on desktop and mobile devices
- **Pure Vanilla JavaScript**: No frameworks, just clean JavaScript
- **Smooth Animations**: Engaging card flip animations and transitions

## Demo

Visit the live demo: **[og-meme-memory.vercel.app](https://og-meme-memory.vercel.app/)**

## How to Play

1. Click on any card to reveal the meme
2. Click on a second card to find its match
3. If the cards match, they stay face up
4. If they don't match, they flip back over
5. Remember the positions and match all pairs to win
6. Try to complete the game in as few moves as possible

## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/achille010/mind-game.git
cd mind-game
```

2. Open `index.html` in your browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Or simply drag and drop `index.html` into your browser.

### Local Development

No build process required! Simply edit the files and refresh your browser:

- `index.html` - HTML structure
- `style.css` - Styling and animations
- `script.js` - Game logic
